CL Timemeter

версия 1.0 (portable)

CL-Timemeter - простое приложение с функцией секундомера.

Функции и управление

Для запуска приложения используйте файл "CL-timemeter.exe" в папке приложения.

Стартовое окно приложения

Появляется при запуске приложения.

Секундомер запущен:

При работе приложения происходит смена цвета фона каждые 10 сеунд.

Секундомер приостановлен:

При постановке секундомера на паузу окно приложения перестает менять цвет фона пока секундомер снова не будет запущен.

Дополнительные функции приложения:

В основном режиме приложения можно также включить вывод настоящего времени в окне секундомера - кнопка с изображением часов.
